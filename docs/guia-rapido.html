Guia Rápido do SiteGen

Este é um guia rápido para começar a usar o SiteGen em 5 minutos.

Instalação

  1. Clone o repositório:

    git clone https://github.com/sitegen/sitegen.git
cd sitegen

  2. Compile o projeto:

    go build -o sitegen main.go

Primeiro Site

  1. Crie a estrutura:

    mkdir meu-site
cd meu-site
mkdir content templates

  2. Configure o site (site.toml):

    [site]
title = "Meu Primeiro Site"
description = "Site criado com SiteGen"

[build]
source = "./content"
output = "./dist"
templates = { default = "./templates/layout.html" }

[plugins]
pipeline = ["markdown", "html"]

  3. Crie um template (templates/layout.html):

    <!DOCTYPE html>
<html>
<head>
    <title>{{.Title}}</title>
</head>
<body>
    <h1>{{.Title}}</h1>
    {{.Content}}
</body>
</html>

  4. Adicione conteúdo (content/index.md):

    ---
title: "Página Inicial"
---

# Bem-vindo!

Este é meu primeiro site com SiteGen.

  5. Gere o site:

    ../sitegen build .

Estrutura de Diretórios

meu-site/
├── site.toml              # Configuração
├── content/               # Conteúdo markdown
│   ├── index.md          # Página inicial
│   ├── sobre.md          # Página sobre
│   └── blog/             # Posts do blog
│       ├── post1.md
│       └── post2.md
├── templates/             # Templates HTML
│   └── layout.html
└── dist/                 # Saída gerada

Recursos Avançados

Metadados YAML

---
title: "Meu Post"
date: "2024-01-15"
category: "blog"
template: "blog"
tags: ["tag1", "tag2"]
draft: false
---

Múltiplos Templates

[build]
templates = { 
    default = "./templates/layout.html", 
    blog = "./templates/blog.html", 
    page = "./templates/page.html" 
}

Recursividade

O SiteGen agora suporta subdiretórios automaticamente:

  • content/blog/post.md → dist/blog/post.html
  • content/docs/guia.md → dist/docs/guia.html

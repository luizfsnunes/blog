Bem-vindo ao SiteGen

Outra mudanca

SiteGen é um gerador de sites estáticos simples e rápido, escrito em Go. Este é um site de exemplo que demonstra as principais funcionalidades do sistema.

Características Principais

  • Simples: Fácil de usar e configurar
  • Rápido: Geração rápida de sites estáticos
  • Extensível: Sistema de plugins modular
  • TOML: Configuração em TOML por site

Como Usar

  1. Crie um diretório para seu site
  2. Adicione um arquivo site.toml com a configuração
  3. Crie a pasta content/ com arquivos markdown
  4. Crie a pasta templates/ com templates HTML
  5. Execute o gerador
./sitegen build ./meu-site

Sobre o SiteGen

SiteGen é um gerador de sites estáticos desenvolvido em Go, focado em simplicidade e performance.

História do Projeto

O SiteGen nasceu da necessidade de ter um gerador de sites estáticos simples, rápido e extensível. Diferente de outras soluções mais complexas, o SiteGen mantém a simplicidade como prioridade.

Arquitetura

O sistema é baseado em três tipos de plugins:

  1. Source Plugins: Leem conteúdo (markdown, JSON, etc.)
  2. Transform Plugins: Processam conteúdo (markdown → HTML)
  3. Output Plugins: Geram saída final (HTML, CSS, assets)

Pipeline de Processamento

Content Files → Source Plugins → Transform Plugins → Output Plugins → Final Site

Tecnologias Utilizadas

  • Go: Linguagem principal do projeto
  • Goldmark: Parser de Markdown
  • TOML: Formato de configuração
  • HTML Templates: Sistema de templates nativo do Go

Vantagens

  • Performance: Processamento paralelo de arquivos
  • Simplicidade: Configuração mínima necessária
  • Extensibilidade: Sistema de plugins modular
  • Flexibilidade: Suporte a múltiplos templates

Teste de Template Layout

Esta página especifica explicitamente o template layout no frontmatter.

Como Funciona

  1. Template obrigatório: Esta página tem template: "layout" no frontmatter
  2. Sistema simples: Sem hierarquia de prioridade, sem fallback
  3. Resultado: Esta página usa o template layout.html

Verificação

Se você está vendo esta página com o layout padrão, significa que o sistema está funcionando corretamente.

Estrutura do Frontmatter

---
title: "Teste de Template Layout"
date: "2024-01-20"
category: "teste"
template: "layout"  # ← Template obrigatório!
tags: ["teste", "layout", "obrigatorio"]
---

Sistema Simplificado

Agora o sistema é muito mais simples:

  • ✅ Template sempre obrigatório no frontmatter
  • ✅ Sem hierarquia de prioridade confusa
  • ✅ Erro claro se não houver template
  • ✅ Lista de templates disponíveis no erro

Comparação

  • Esta página: template: "layout" → usa template layout
  • Páginas do blog: template: "blog" → usa template blog
  • Páginas normais: template: "page" → usa template page
  • Sem template: ❌ Erro claro e útil

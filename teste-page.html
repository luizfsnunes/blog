Teste de Template Page

Esta página ESPECIFICA explicitamente o template page no frontmatter, então deve usar o template page.

Como Funciona

  1. Template no frontmatter: Esta página tem template: "page" no frontmatter
  2. Prioridade alta: O sistema usa o template especificado
  3. Resultado: Esta página usa o template page.html

Verificação

Se você está vendo esta página com o layout de página (com cabeçalho de página, estrutura específica), significa que a prioridade do frontmatter está funcionando corretamente.

Estrutura do Frontmatter

---
title: "Teste de Template Page"
date: "2024-01-20"
category: "teste"
template: "page"  # ← Template específico
tags: ["teste", "page", "template"]
---

Características do Template Page

Este template deve ter:

  • Layout para páginas estáticas
  • Cabeçalho de página
  • Estrutura de conteúdo
  • Navegação principal

Hierarquia de Prioridade

  1. Prioridade Alta: template: "page" no frontmatter
  2. Prioridade Baixa: Template padrão do TOML (fallback)

Teste Completo

Agora você pode comparar:

  • teste-layout.md: Sem template → usa fallback
  • teste-blog.md: template: "blog" → usa blog
  • teste-page.md: template: "page" → usa page

Todos os três cenários devem funcionar corretamente!

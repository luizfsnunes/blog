Teste Sem Template

Esta página NÃO especifica um template no frontmatter, então deve gerar um erro claro.

Como Funciona

  1. Sem template no frontmatter: Esta página não tem template: "algo" no frontmatter
  2. Erro obrigatório: O sistema deve dar erro claro e útil
  3. Resultado esperado: Build falha com mensagem explicativa

Estrutura do Frontmatter

---
title: "Teste Sem Template"
date: "2024-01-20"
category: "teste"
tags: ["teste", "erro", "sem-template"]
# Note: NÃO há template: "algo" aqui!
---

Erro Esperado

O sistema deve gerar um erro como:

page 'example-site/content/teste-sem-template.md' must specify a template in frontmatter (e.g., template: 'layout')

Como Corrigir

Para corrigir, adicione um template ao frontmatter:

---
title: "Teste Sem Template"
date: "2024-01-20"
category: "teste"
template: "layout"  # ← Adicionar esta linha
tags: ["teste", "erro", "sem-template"]
---

Vantagens do Sistema Obrigatório

  • ✅ Clareza: Sempre fica claro qual template será usado
  • ✅ Debug fácil: Erro específico e útil
  • ✅ Consistência: Todas as páginas seguem o mesmo padrão
  • ✅ Simplicidade: Sem lógica complexa de fallback

Teste Real

Para testar o erro real, remova a linha template: "layout" deste arquivo e execute o build novamente. Você verá o erro claro e útil!
